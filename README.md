# Adit's Personal Portfolio 🚀

Welcome to the source code of my personal portfolio! I am an Information Technology graduate and Full-Stack Developer passionate about building high-converting, scalable, and visually engaging web applications.

## 🌟 Key Highlights
- **Modern UI/UX:** Built with a dark-themed glassmorphism aesthetic.
- **Smooth Animations:** Powered by Framer Motion for seamless page transitions and interactive elements.
- **Thesis Showcase:** Prominently features my academic research project, **SiKeuMas** (Web-Based Mosque Financial Management System using Cash Basis Accounting).
- **Direct Contact:** Integrated Floating WhatsApp action button for instant communication.

## 🛠️ Tech Stack
- **Framework:** React 19 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Routing:** React Router v7
- **Animations:** Framer Motion

## 💻 Local Setup
To run this project locally on your machine:
1. Clone the repository: `git clone https://github.com/Wilhelm-art/portofolio-adit.git`
2. Navigate into the directory: `cd portofolio-adit`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
